# Contributing to LearnSphere

Thanks for your interest in contributing.

## Good first contributions

- Improve README or documentation
- Add trusted catalog entries
- Improve UI responsiveness
- Improve search result ranking
- Add tests
- Fix API errors

## Development

```bash
npm install
cp .env.example .env
npm run dev
```

## Rules

- Do not add pirated PDF links.
- Do not commit real API keys or secrets.
- Prefer official, legal, public-domain, library, or preview sources.
